## 项目简介

这是一个模拟小型低轨试验卫星地面监控系统的Web应用，采用前端原生技术栈开发，实现了卫星信息管理、实时状态监控、指令下发和历史数据查询的完整业务闭环。

## 核心功能

### 1. 卫星基础信息管理
- ✅ 新增卫星（编号唯一性校验）
- ✅ 查询卫星（支持模糊搜索）
- ✅ 修改卫星（在线状态限制）
- ✅ 删除卫星（级联删除关联数据）

### 2. 卫星实时状态监控
- ✅ 模拟数据自动生成（每5秒）
- ✅ 实时数据刷新（5秒间隔）
- ✅ 多卫星切换监控
- ✅ 可视化展示（温度、电量、信号强度、角度罗盘、趋势图）

### 3. 卫星指令下发
- ✅ 指令下发（调整角度、休眠、唤醒）
- ✅ 状态校验（离线卫星限制）
- ✅ 指令跟踪（执行状态更新）
- ✅ 指令记录查询

### 4. 历史数据查询与导出
- ✅ 条件查询（时间范围限制24小时）
- ✅ 数据表格展示
- ✅ 趋势图可视化
- ✅ CSV文件导出

## 技术栈

- **前端框架**: 原生HTML5 + CSS3 + JavaScript
- **UI框架**: Bootstrap 5.3
- **图表库**: ECharts 5.4
- **图标库**: Font Awesome 6.4
- **数据存储**: 浏览器localStorage（模拟数据库）

## 运行说明

### 环境要求
- 现代浏览器（Chrome 90+、Firefox 88+、Safari 14+）
- 支持ES6+ JavaScript
- 支持localStorage

### 启动步骤
1. 克隆项目到本地
2. 使用浏览器打开 `satellite-system/index.html`
3. 开始使用系统

### 功能演示
1. **基础信息管理**: 点击"基础信息管理"进入，尝试新增卫星
2. **实时监控**: 点击"实时状态监控"查看卫星状态和图表
3. **指令下发**: 选择卫星下发调整角度指令，观察状态变化
4. **数据查询**: 查询历史数据并导出CSV文件

## 数据模拟说明

- **卫星数据**: 预置3颗卫星（SAT001-SAT003）
- **遥测数据**: 每5秒自动生成，保留最近1天数据
- **指令记录**: 支持调整角度、休眠、唤醒三种指令
- **数据持久化**: 使用浏览器localStorage存储，重启浏览器数据保留

## 核心特性

### 实时性
- 遥测数据每5秒自动更新
- 前端页面实时刷新显示
- 指令执行状态实时跟踪

### 可视化
- 温度、电量、信号强度数字显示
- 信号强度圆形仪表盘
- 角度罗盘实时指向
- 温度/电量趋势折线图

### 交互性
- 响应式设计，支持移动端
- 表单实时校验
- 操作成功/失败提示
- 数据导出功能

### 业务完整性
- 卫星状态影响操作权限
- 指令执行有时间延迟模拟
- 数据关联完整性保证

## 注意事项

1. **数据持久化**: 数据存储在浏览器localStorage中，清除浏览器数据会丢失
2. **性能考虑**: 模拟数据量有限，实际生产环境需后端数据库支持
3. **浏览器兼容**: 建议使用Chrome浏览器获得最佳体验
4. **数据范围**: 温度-20~60℃，电量0~100%，角度-90~90°

## 开发说明

本项目采用前端纯静态方式实现，无需服务器环境。所有业务逻辑通过JavaScript模拟，数据通过localStorage持久化。主要用于学习和演示目的，展示了现代Web前端技术在复杂业务场景下的应用能力。
